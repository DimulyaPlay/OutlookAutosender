Программа для автоматической отправки сообщений с вложениями через Outlook.
Предусмотрена работа по расписанию и по таймеру, а главное, учтены требования для обмена с Росреестром (опциональны):
1. Сбор только тех документов, которые имеют файл отсоединенной подписи(.sig);
2. Архивирование вложений;
3. Шифрование архива заданным сертификатом из хранилища личных сертификатов.

Программа при работе в автоматических режимах не требует запуска Outlook, через MAPI подключается к аккаунту и отправляет письма.
Все отправленные пакеты также будут находиться в папке Отправленные/Sent items.
Также предусмотрен запуск нескольких экземпляров для запуска с разными настройками, к примеру: 
одна программа отправляет документы, для которых не требуется .sig файл, а другая отправляет только подписанные документы, 
затем отправленные обеими экземплярами программы пакеты складываются в одну папку "архив".
Имеется система логгирования, которая записывает в файл какие архивы в какое время были отправлены и какие файлы они содержат.

Особенности работы:
-Программа берет только файлы из указанной папки, в папки она не заглядывает, соответственно, папка для отправленных архивов может находиться
прямо в папке для отправки.
-Для ручной отправки (кнопка "Отправить вручную сейчас") нужно запускать Outlook, так как она просто формирует письмо, а отпавляет уже сам пользователь.
-Кнопка свернуть скрывает окно в трей, закрыть закрывает ее полностью.

Системные требования:
Windows 7 или более поздние версии (Для работы на win 7 необходимо наличие обновлений безопасности из папки redist)
Outlook 2010 или более поздние версии (Ранние версии не тестировались)
КриптоПро CSP 4.0.99 или более поздние версии (Только если планируется отправка зашифрованных архивов в Росреестр)
